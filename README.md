## Monemon demo app instructions

Download the app from Testflight. Testflight can be downloaded from the App Store. The phone (and Testflight) should be logged in with the same Apple ID, that you provided for the access.

Always use the latest version, if not explicitly specified otherwise.

Once the app has been downloaded, log in to the app using the same Apple ID (e-mail) you provided for the testing. When logging in use the first button (Smart-ID) to log in. There is no authentication, you will be automatically logged in.

As this is a demo app, all features may not always work (cards), but the look and feel is exactly as the production app. The only difference is in the App Icon, which is red instead of the usual purple gradient.

### Wrapped generation
To use the Wrapped demo feature long press on the butterfly icon next to the `Wrapped` widget title on the Home screen. 

<img src="https://mowomedia.s3.eu-north-1.amazonaws.com/home_screen.jpeg" alt="drawing" width="200"/>

This will open a modal, where you will see 2 tabs - `Generate` and `Previous`

<img src="https://mowomedia.s3.eu-north-1.amazonaws.com/modal_1.jpeg" alt="drawing2" width="200"/>
<img src="https://mowomedia.s3.eu-north-1.amazonaws.com/modal_2.jpeg" alt="drawing3" width="200"/>

On the first tab there is a text box, where you can specify a character. The more info you input about the profile and last week, the more detailed Wrapped you will receive. Play around with it a few times to get a feel for it. Once the profile text is ready, pressing `Generate` will start the Wrapped generation. This may take up to a minute, so be patient. Once the generation is ready, you may close the modal and open the wrapped by pressing the wrapped card. This will start the Wrapped slideshow, which is usually 5-7 slides long.

NB! The Wrapped content is generated by a LLM, so it may be inaccurate with numbers and hallucinate at times. Usually trying again with a modified prompt will fix the issues. If any issues persist after multiple generations, let us know, we will find a fix.

On the second tab you will see the history of your previously generated Wrapped profiles. Clicking on them will load them to the Wrapped widget and you can replay the same Wrapped slides you generated earlier.


### Mission generation
To use the Mission demo feature enable Save mode from the top left corner. 
Then either click Generate (when using for the first time) or long press `Missions` widget title on the screen.

<img src="https://mowomedia.s3.eu-north-1.amazonaws.com/modal_3.png" alt="drawing2" width="200"/>

This will open a similar modal, where you will see 2 tabs - `Generate` and `Previous`

<img src="https://mowomedia.s3.eu-north-1.amazonaws.com/modal_4.png" alt="drawing3" width="200"/>

On the first tab there are three text boxes. The first one allows you to describe the mission, the second one is for the limit and the third one sets how much the user has already spent. The more info you input to the first description the more detailed mission details you will get. Play around with it a few times to get a feel for it. Once the profile text is ready, pressing `Generate` will start the Mission generation. This may take up to a minute, so be patient. Once the generation is ready, you may close the modal and the mission is shown in the widget. Pressing on the widget will open up mission details, where you can see a description and the contributing transactions.

Above the mission widget a spending treemap will appear, showing a generated weekly category breakdown based on the mission topic. For example, a mission about dining out will show Restaurants & Accommodation as the prominent category. 

On the second tab you will see the history of your previously generated Missions. Clicking on them will load them to the Mission widget.

### Homescreen widget

The homescreen widget is only available on iOS.

To add the widget to your homescreen:
1. Long press on the homescreen until the icons start jiggling
2. Tap the `Edit` button in the top left corner
3. Search for **Monemon Dev** and select it
4. Choose either the small or medium size and tap **Add Widget**

The widget displays your active weekly mission - remaining budget, mission title, progress bar, estimated savings, and days left until the end of the challenge.

**What the widget shows:**
- If you are not logged in to the app: "Open Monemon to sign in"
- If you have an active mission: mission data
- The widget updates automatically when you open the app or return to it from the background
